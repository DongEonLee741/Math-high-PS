덧셈정리 기초 확인

기초 UP 문제 1

두 사건 \(A\), \(B\)에 대하여 \(P(A)=\dfrac{1}{3}\), \(P(B)=\dfrac{1}{4}\), \(P(A \cap B)=\dfrac{1}{6}\)일 때, \(P(A \cup B)\)를 구하시오.

▶ 정답 확인
정답

\(\displaystyle \frac{5}{12}\)

풀이

\(\displaystyle P(A \cup B) = P(A) + P(B) - P(A \cap B) = \frac{1}{3} + \frac{1}{4} - \frac{1}{6} = \frac{4+3-2}{12} = \frac{5}{12}\)

기초 UP 문제 2

\(P(A)=\dfrac{1}{2}\), \(P(A \cap B)=\dfrac{1}{6}\), \(P(A \cup B)=\dfrac{2}{3}\)일 때, \(P(B)\)를 구하시오.

▶ 정답 확인
정답

\(\displaystyle \frac{1}{3}\)

풀이

\(\displaystyle \frac{2}{3} = \frac{1}{2} + P(B) - \frac{1}{6}\)  →  \(\displaystyle P(B) = \frac{2}{3} - \frac{1}{2} + \frac{1}{6} = \frac{4-3+1}{6} = \frac{1}{3}\)

기초 UP 문제 3

두 사건 \(A\), \(B\)가 서로 배반사건이고 \(P(A \cup B)=\dfrac{3}{5}\), \(P(A):P(B)=2:1\)일 때, \(P(A)\)를 구하시오.

▶ 정답 확인
정답

\(\displaystyle \frac{2}{5}\)

풀이

배반사건이므로 \(P(A \cup B) = P(A) + P(B)\).
\(P(A) = 2k\), \(P(B) = k\)로 놓으면 \(2k + k = \dfrac{3}{5}\), \(k = \dfrac{1}{5}\).
따라서 \(P(A) = \dfrac{2}{5}\).

기초 UP 문제 4

주머니 안에 1부터 20까지의 숫자가 하나씩 적힌 20개의 공이 들어 있다. 이 주머니에서 임의로 한 개의 공을 꺼낼 때, 이 공에 적힌 수가 4의 배수일 사건을 \(A\), 6의 배수일 사건을 \(B\)라 하자. 다음을 구하시오.

⑴

\(P(A)\)

▶ 정답 확인
정답

\(\displaystyle \frac{1}{4}\)

풀이

4의 배수: 4, 8, 12, 16, 20 → 5개. \(P(A) = \dfrac{5}{20} = \dfrac{1}{4}\)

⑵

\(P(B)\)

▶ 정답 확인
정답

\(\displaystyle \frac{3}{20}\)

풀이

6의 배수: 6, 12, 18 → 3개. \(P(B) = \dfrac{3}{20}\)

⑶

\(P(A \cap B)\)

▶ 정답 확인
정답

\(\displaystyle \frac{1}{20}\)

풀이

4와 6의 공배수(12의 배수): 12 → 1개. \(P(A \cap B) = \dfrac{1}{20}\)

⑷

\(P(A \cup B)\)

▶ 정답 확인
정답

\(\displaystyle \frac{7}{20}\)

풀이

\(\displaystyle P(A \cup B) = \frac{1}{4} + \frac{3}{20} - \frac{1}{20} = \frac{5+3-1}{20} = \frac{7}{20}\)

스스로 확인하기 문제 1

두 사건 \(A\), \(B\)에 대하여 \(P(A)=\dfrac{1}{2}\), \(P(B)=\dfrac{1}{4}\), \(P(A \cup B)=\dfrac{1}{3}\)일 때, \(P(A \cap B)\)를 구하시오.

▶ 정답 확인
정답

\(\displaystyle \frac{5}{12}\)

풀이

\(\displaystyle P(A \cap B) = P(A) + P(B) - P(A \cup B) = \frac{1}{2} + \frac{1}{4} - \frac{1}{3} = \frac{6+3-4}{12} = \frac{5}{12}\)

스스로 확인하기 문제 2

1, 2, 3, 4, 5의 숫자가 하나씩 적힌 5장의 카드에서 3장을 뽑아 만드는 세 자리 자연수 중에서 짝수일 확률을 구하시오.

▶ 정답 확인
정답

\(\displaystyle \frac{2}{5}\)

풀이

전체: \({}_{5}P_{3} = 60\)가지.
짝수 → 일의 자리가 2 또는 4 (2가지), 나머지 2자리는 남은 4개 중 택2 배열 \(= {}_{4}P_{2} = 12\).
짝수: \(2 \times 12 = 24\). 확률 \(= \dfrac{24}{60} = \dfrac{2}{5}\).

스스로 확인하기 문제 3

어느 반 학생 전체를 대상으로 조사한 결과, 영화 A를 본 학생은 30%, 영화 B를 본 학생은 45%, 두 영화를 모두 본 학생은 10%였다. 이 반 학생 중 임의로 한 명을 선택할 때, 영화 A 또는 영화 B를 본 학생일 확률을 구하시오.

▶ 정답 확인
정답

0.65

풀이

\(P(A \cup B) = P(A) + P(B) - P(A \cap B) = 0.30 + 0.45 - 0.10 = 0.65\)
