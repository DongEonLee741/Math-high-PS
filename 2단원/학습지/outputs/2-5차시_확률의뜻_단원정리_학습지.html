문제 1 4개의 숫자 1, 2, 3, 4가 하나씩 적힌 카드를 모두 사용하여 네 자리 자연수를 만들 때, 이 수가 짝수일 확률을 구하시오.
▶ 풀이 보기
풀이
전체 경우의 수: 4개의 숫자를 모두 나열 → \(4! = 24\)

짝수가 되려면 일의 자리가 2 또는 4이어야 한다.
• 일의 자리 선택: 2가지 (2 또는 4)
• 나머지 3자리 배열: \(3! = 6\)가지
• 짝수인 경우의 수: \(2 \times 3! = 12\)

\(\therefore\; P = \dfrac{12}{24} = \boldsymbol{\dfrac{1}{2}}\)

문제 3 여학생 3명과 남학생 4명 중에서 임의로 4명의 대표를 뽑을 때, 여학생과 남학생의 수가 같을 확률을 구하시오.
▶ 풀이 보기
풀이
여학생과 남학생의 수가 같으려면 각각 2명씩 뽑아야 한다.

• 전체: \({}_{7}\mathrm{C}_{4} = 35\)
• 여학생 2명, 남학생 2명: \({}_{3}\mathrm{C}_{2} \times {}_{4}\mathrm{C}_{2} = 3 \times 6 = 18\)

\(\therefore\; P = \dfrac{18}{35} = \boldsymbol{\dfrac{18}{35}}\)

문제 5 주머니 안에 1부터 9까지의 자연수가 하나씩 적힌 9개의 공이 들어 있다. 이 주머니에서 임의로 3개의 공을 동시에 꺼낼 때, 꺼낸 공에 적힌 세 수의 합이 홀수일 확률을 구하시오.
▶ 풀이 보기
풀이
홀수 5개: \(\{1, 3, 5, 7, 9\}\), 짝수 4개: \(\{2, 4, 6, 8\}\)

세 수의 합이 홀수가 되려면:
• (홀수 1개) + (짝수 2개): \({}_{5}\mathrm{C}_{1} \times {}_{4}\mathrm{C}_{2} = 5 \times 6 = 30\)
• (홀수 3개): \({}_{5}\mathrm{C}_{3} = 10\)
• 합이 홀수인 경우의 수: \(30 + 10 = 40\)

전체: \({}_{9}\mathrm{C}_{3} = 84\)

\(\therefore\; P = \dfrac{40}{84} = \boldsymbol{\dfrac{10}{21}}\)
문제 6 수학으로 생각 넓히기

수빈, 지원, 선아가 차례로 한 개의 주사위를 한 번씩 던져서 각자 나온 눈의 수만큼 말판에서 말을 옮기는 게임을 하려고 한다. 세 명의 말이 출발 위치에 있는 상태에서 한 번씩 주사위를 던졌을 때, 지원이가 놓은 말이 수빈이와 선아가 놓은 말 사이에 있게 될 확률을 구해 보자.

▶ 풀이 보기
풀이
전체 경우의 수: \(6^3 = 216\)

수빈, 지원, 선아의 눈이 모두 다른 경우:
\(6 \times 5 \times 4 = 120\)가지

이 중 지원이가 가운데인 경우는 대칭성에 의해
\(\dfrac{120}{3} = 40\)가지

\(\therefore\; P = \dfrac{40}{216} = \boldsymbol{\dfrac{5}{27}}\)

문제 9 상자 속에 흰 공 5개와 검은 공 3개가 들어 있다. 이 상자에서 임의로 3개의 공을 동시에 꺼낼 때, 다음을 구하시오.
⑴ 흰 공을 3개 꺼낼 확률
⑵ 흰 공을 1개, 검은 공을 2개 꺼낼 확률
▶ 풀이 보기
풀이
전체: \({}_{8}\mathrm{C}_{3} = 56\)

⑴ 흰 공 3개: \(\dfrac{{}_{5}\mathrm{C}_{3}}{{}_{8}\mathrm{C}_{3}} = \dfrac{10}{56} = \boldsymbol{\dfrac{5}{28}}\)

⑵ 흰 1 + 검 2: \(\dfrac{{}_{5}\mathrm{C}_{1} \times {}_{3}\mathrm{C}_{2}}{{}_{8}\mathrm{C}_{3}} = \dfrac{5 \times 3}{56} = \boldsymbol{\dfrac{15}{56}}\)
문제 10 서로 다른 두 개의 주사위를 동시에 던질 때, 다음을 구하시오.
⑴ 나오는 두 눈의 수가 서로 같을 확률
⑵ 나오는 두 눈의 수의 합이 12일 확률
⑶ 나오는 두 눈의 수의 곱이 0일 확률
▶ 풀이 보기
풀이
전체: \(6 \times 6 = 36\)

⑴ 같은 눈: \((1,1),(2,2),\ldots,(6,6)\) → 6가지
\(P = \dfrac{6}{36} = \boldsymbol{\dfrac{1}{6}}\)

⑵ 합이 12: \((6,6)\)뿐 → 1가지
\(P = \boldsymbol{\dfrac{1}{36}}\)

⑶ 곱이 0: 주사위 눈은 1~6이므로 0이 나올 수 없다.
\(P = \boldsymbol{0}\) (불가능 사건)
문제 11 표본공간 \(S\)의 세 사건 \(A, B, C\)에 대하여 \(A \cap C = \varnothing\), \(B \subset C\)가 성립할 때, 보기에서 옳은 것을 있는 대로 고르시오.
ㄱ. \(A \cup C = S\)
ㄴ. \(A\)와 \(B\)는 서로 배반사건이다.
ㄷ. \(A\)는 \(C\)의 여사건이다.
▶ 풀이 보기
풀이
답: ㄴ

ㄱ. \(A \cap C = \varnothing\)이지만 \(A \cup C = S\)가 반드시 성립하지는 않는다.
    예) \(S=\{1,2,3\}, A=\{1\}, C=\{2\}\)이면 \(A \cup C = \{1,2\} \neq S\)
    → 거짓

ㄴ. \(B \subset C\)이고 \(A \cap C = \varnothing\)이므로
    \(A \cap B \subset A \cap C = \varnothing\) → \(A \cap B = \varnothing\)
    따라서 \(A\)와 \(B\)는 서로 배반사건이다. → 참

ㄷ. \(A\)가 \(C\)의 여사건이려면 \(A \cap C = \varnothing\)이고 \(A \cup C = S\)이어야 하는데,
    \(A \cup C = S\)는 보장되지 않는다. → 거짓
