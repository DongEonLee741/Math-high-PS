## 3. 교과서 원문 텍스트

> 상세 원문은 `교과서_전사본_p53_기초UP.md`를 참조.
> "교과서 원문 그대로"로 표시된 부분은 전사본의 해당 구간을 충실히 반영할 것.

### 섹션 1: 덧셈정리 기초 확인

#### 기초 UP 문제 1 (교과서 원문 그대로)
두 사건 $A$, $B$에 대하여 $P(A) = \frac{1}{3}$, $P(B) = \frac{1}{4}$, $P(A \cap B) = \frac{1}{6}$일 때, $P(A \cup B)$를 구하시오.

**정답**: $\frac{5}{12}$

**풀이**: $P(A \cup B) = P(A) + P(B) - P(A \cap B) = \frac{1}{3} + \frac{1}{4} - \frac{1}{6} = \frac{4+3-2}{12} = \frac{5}{12}$

#### 기초 UP 문제 2 (교과서 원문 그대로)
두 사건 $A$, $B$에 대하여 $P(A) = \frac{1}{2}$, $P(A \cap B) = \frac{1}{6}$, $P(A \cup B) = \frac{2}{3}$일 때, $P(B)$를 구하시오.

**정답**: $\frac{1}{3}$

**풀이**: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$에서 $\frac{2}{3} = \frac{1}{2} + P(B) - \frac{1}{6}$ → $P(B) = \frac{2}{3} - \frac{1}{2} + \frac{1}{6} = \frac{4-3+1}{6} = \frac{1}{3}$

#### 기초 UP 문제 3 (교과서 원문 그대로)
두 사건 $A$, $B$가 서로 배반사건이고 $P(A \cup B) = \frac{3}{5}$, $P(A):P(B) = 2:1$일 때, $P(A)$를 구하시오.

**정답**: $\frac{2}{5}$

**풀이**: 배반이므로 $P(A \cup B) = P(A) + P(B)$. $P(A) = 2k$, $P(B) = k$로 놓으면 $3k = \frac{3}{5}$ → $k = \frac{1}{5}$ → $P(A) = \frac{2}{5}$

#### 기초 UP 문제 4 (교과서 원문 그대로)
주머니 안에 1부터 20까지의 숫자가 하나씩 적힌 20개의 공이 들어 있다. 이 주머니에서 임의로 한 개의 공을 꺼낼 때, 이 공에 적힌 수가 4의 배수일 사건을 $A$, 6의 배수일 사건을 $B$라 하자. 다음을 구하시오.
⑴ $P(A)$ ⑵ $P(B)$ ⑶ $P(A \cap B)$ ⑷ $P(A \cup B)$

**정답**: ⑴ $\frac{1}{4}$ ⑵ $\frac{3}{20}$ ⑶ $\frac{1}{20}$ ⑷ $\frac{7}{20}$

**풀이**:
- $A = \{4, 8, 12, 16, 20\}$ → $n(A) = 5$ → $P(A) = \frac{5}{20} = \frac{1}{4}$
- $B = \{6, 12, 18\}$ → $n(B) = 3$ → $P(B) = \frac{3}{20}$
- $A \cap B = \{12\}$ → $P(A \cap B) = \frac{1}{20}$
- $P(A \cup B) = \frac{1}{4} + \frac{3}{20} - \frac{1}{20} = \frac{5+3-1}{20} = \frac{7}{20}$

#### 스스로 확인하기 문제 1 (교과서 원문 그대로)
두 사건 $A$, $B$에 대하여 $P(A) = \frac{1}{2}$, $P(B) = \frac{1}{4}$, $P(A \cup B) = \frac{1}{3}$일 때, $P(A \cap B)$를 구하시오.

**정답**: $\frac{5}{12}$

**풀이**: $P(A \cap B) = P(A) + P(B) - P(A \cup B) = \frac{1}{2} + \frac{1}{4} - \frac{1}{3} = \frac{6+3-4}{12} = \frac{5}{12}$

#### 스스로 확인하기 문제 2 (교과서 원문 그대로)
5개의 숫자 1, 2, 3, 4, 5 중에서 임의로 서로 다른 3개의 숫자를 택하여 만든 세 자리 자연수가 짝수일 확률을 구하시오.

**정답**: $\frac{2}{5}$

**풀이**:
- 전체: ${}_{5}P_{3} = 60$
- 짝수 → 일의 자리 2 또는 4 (2가지), 나머지 2자리 ${}_{4}P_{2} = 12$
- 짝수인 경우: $2 \times 12 = 24$
- 확률: $\frac{24}{60} = \frac{2}{5}$

#### 스스로 확인하기 문제 3 (교과서 원문 그대로)
민수네 반 학생 중 A 영화를 본 학생은 30%, B 영화를 본 학생은 45%이고 두 영화를 모두 본 학생은 10%라 한다. 민수네 반 학생 중에서 임의로 한 명을 뽑을 때, A 영화 또는 B 영화를 본 학생일 확률을 구하시오.

**정답**: $0.65$

**풀이**: $P(A \cup B) = 0.30 + 0.45 - 0.10 = 0.65$

### 섹션 2: 조합과 확률의 연결

#### 기초 UP 문제 5 (교과서 원문 그대로)
4개의 당첨 제비가 포함되어 있는 10개의 제비 중에서 임의로 2개의 제비를 동시에 뽑을 때, 다음을 구하시오.
⑴ 2개 모두 당첨 제비가 아닐 확률
⑵ 적어도 한 개가 당첨 제비일 확률

**정답**: ⑴ $\frac{1}{3}$ ⑵ $\frac{2}{3}$

**풀이**:
- ⑴ 전체: ${}_{10}C_{2} = 45$, 당첨 아닌 제비 6개에서 2개: ${}_{6}C_{2} = 15$ → $\frac{15}{45} = \frac{1}{3}$
- ⑵ 여사건: $1 - \frac{1}{3} = \frac{2}{3}$

#### 스스로 확인하기 문제 4 (교과서 원문 그대로)
깨가 들어 있는 송편 6개와 콩이 들어 있는 송편 8개 중에서 임의로 3개의 송편을 택할 때, 적어도 한 개에는 깨가 들어 있을 확률을 구하시오.

**정답**: $\frac{11}{13}$

**풀이**:
- 전체: ${}_{14}C_{3} = 364$
- 여사건(깨 0개 = 콩만 3개): ${}_{8}C_{3} = 56$
- $P(\text{여사건}) = \frac{56}{364} = \frac{2}{13}$
- $P(\text{적어도 깨 1개}) = 1 - \frac{2}{13} = \frac{11}{13}$

#### 스스로 확인하기 문제 5 (교과서 원문 그대로)
빨간 장미꽃 7송이와 노란 장미꽃 5송이 중에서 임의로 5송이의 장미꽃을 택하여 꽃다발을 만들 때, 노란 장미꽃이 3송이 이하로 포함될 확률을 구하시오.

**정답**: $\frac{21}{22}$

**풀이**:
- 전체: ${}_{12}C_{5} = 792$
- 여사건(노란 4송이 이상):
  - 노란 4 + 빨간 1: ${}_{5}C_{4} \times {}_{7}C_{1} = 5 \times 7 = 35$
  - 노란 5 + 빨간 0: ${}_{5}C_{5} \times {}_{7}C_{0} = 1 \times 1 = 1$
  - 합: $36$
- $P(\text{여사건}) = \frac{36}{792} = \frac{1}{22}$
- $P(\text{노란 3이하}) = 1 - \frac{1}{22} = \frac{21}{22}$

### 섹션 3: 수학으로 생각 넓히기

#### 생일 문제 (교과서 원문 그대로)
5명의 학생 승준, 예진, 철수, 미나, 은호의 생일은 모두 9월에 있다. 이 5명의 학생 중에서 적어도 2명의 생일이 같을 확률을 다음 단계에 따라 구해 보자.

- **1단계**: 5명의 학생의 생일이 모두 다를 확률을 구한다.
- **2단계**: 여사건을 이용하여 적어도 2명의 생일이 같을 확률을 구한다.

**정답**:
- $P(A^c) = \frac{{}_{30}P_{5}}{30^5} = \frac{30 \times 29 \times 28 \times 27 \times 26}{30^5} = \frac{17100720}{24300000} = \frac{2639}{3750}$
- $P(A) = 1 - \frac{2639}{3750} = \frac{1111}{3750} \approx 0.296$

### 5.1 STEP 빈칸 채우기 (섹션 2의 문제 8, 9, 10에 적용)

#### 목적
학생 난점인 "조합(C) 계산과 확률 계산의 연결"에 대한 단계별 즉시 피드백. 분모(전체 경우의 수)와 분자(원하는 경우의 수)가 각각 무엇을 세는 것인지를 STEP 구조로 명시화한다.

#### 교육적 설계 원칙
- **과정 피드백 + 과제 피드백 결합**: Bush(2021) 연구에서 이 결합이 d=0.39 효과 크기를 기록. 단순 정오 판단이 아니라 각 STEP에서 학생의 접근을 확인하고 다음 단계로 연결.
- **즉시 피드백은 해석적으로**: ✓/✗ 대신 학생이 입력한 값과 정답을 나란히 표시. 틀렸을 때 정답을 보여주되, "왜 이 값인지"에 대한 힌트 문장을 함께 제공.
- **인터랙티브 완료 후 필기 영역**: STEP을 모두 완료한 뒤 writing-area에서 전체 풀이를 정리.

| 2 | "'적어도 깨 1개'의 여사건은?" | 선택: ① 깨만 3개 / ② 콩만 3개 / ③ 깨 2개 이상 | ② | "'적어도 하나 깨' ↔ '깨 0개 = 콩만'이 여사건입니다" |
| 3 | "콩 송편 8개에서 3개를 뽑는 경우의 수 $_8C_3$ = ?" | 숫자 입력 | 56 | "$\frac{8 \times 7 \times 6}{3 \times 2 \times 1}$을 계산해 보세요" |
| 4 | "여사건의 확률 = STEP 3 ÷ STEP 1 = ?" | 분수 입력 | 56/364 = 2/13 | "56 ÷ 364를 약분하면?" |
| 5 | "구하는 확률 = 1 − 여사건 확률 = ?" | 분수 입력 | 11/13 | "$1 - \frac{2}{13}$을 계산해 보세요" |

#### UI 동작 — 스스로 확인하기 5번 예시

**"노란 장미꽃이 3송이 이하로 포함될 확률"**

| STEP | 안내 문구 | 입력 | 정답 | 피드백(오답 시) |
|------|----------|------|------|----------------|
| 1 | "전체 12송이에서 5송이를 뽑는 경우의 수 $_{12}C_5$ = ?" | 숫자 입력 | 792 | "$\frac{12 \times 11 \times 10 \times 9 \times 8}{5!}$을 계산해 보세요" |
| 2 | "'노란 3이하'의 여사건은?" | 선택: ① 노란 4 이상 / ② 노란 0개 / ③ 빨간 3 이상 | ① | "'3이하' ↔ '4이상'이 여사건입니다" |
| 3 | "노란 4 + 빨간 1의 경우의 수: $_5C_4 \times _7C_1$ = ?" | 숫자 입력 | 35 | "$5 \times 7 = 35$" |
| 4 | "노란 5 + 빨간 0의 경우의 수: $_5C_5 \times _7C_0$ = ?" | 숫자 입력 | 1 | "$1 \times 1 = 1$" |
| 5 | "여사건 경우의 수 합 = STEP 3 + STEP 4 = ?" | 숫자 입력 | 36 | "$35 + 1 = 36$" |
| 6 | "여사건 확률 = STEP 5 ÷ STEP 1 = ?" | 분수 입력 | 36/792 = 1/22 | "36 ÷ 792를 약분하면?" |
| 7 | "구하는 확률 = 1 − 여사건 확률 = ?" | 분수 입력 | 21/22 | "$1 - \frac{1}{22}$을 계산해 보세요" |

#### 입력 UI 규격

- **숫자 입력**: `<input type="number">`, 모바일에서 숫자 키패드 노출
- **분수 입력**: 분자/분모 각각 `<input type="number">` + 가운데 분수선 (가로줄 시각화). 약분 여부는 검사하지 않음 — 15/45도 정답, 1/3도 정답으로 처리 (수학적 동치 판별: 분자×상대분모 === 분모×상대분자)
- **선택 입력**: 라디오 버튼 3개
- **확인 버튼**: 각 STEP마다 "확인" 버튼. 정답이면 다음 STEP 활성화, 오답이면 피드백 표시 + 재시도 가능 (총 3회 시도 — 최초 1회 + 재시도 2회 — 후 정답 공개)
- **전체 상태**: 모든 STEP 완료 시 "풀이 완성!" 메시지 + 아래에 writing-area 활성화 (전체 풀이 정리용)

#### 예외 처리
- 음수, 0, 소수점 입력 → "자연수를 입력해 주세요" 안내
- 분모에 0 입력 → "분모는 0이 될 수 없습니다" 안내
- STEP 건너뛰기 불가 — 이전 STEP 완료 전 다음 STEP 비활성화

#### 구현 방식
- Vanilla JS (외부 라이브러리 없음)
- 각 문제의 STEP 데이터를 JS 객체 배열로 정의
- 정답은 명세서 §3의 값이 ground truth — Claude Code가 독자적으로 재계산하지 말 것

### 5.2 생일 문제 시뮬레이션 (섹션 3)

#### 목적
직관과 수학적 결과의 괴리를 체험. "5명 중 적어도 2명의 생일이 같을 확률"에 대한 학생의 직관적 예측이 수학적 계산(≈0.296)과 얼마나 다른지를 시뮬레이션으로 확인.

#### 교육적 설계 원칙
- **예측→확인→반성 순환** (Desmos 설계 원칙): 시뮬레이션 실행 전 예측 입력 필수
- **해석적 피드백**: 맞다/틀리다 없이, 예측값과 시뮬레이션 결과를 나란히 표시
- **교과서 2단계 구조 준수**: 1단계(모두 다를 확률) → 2단계(여사건)

#### UI 동작

**Phase 1: 예측**
- 안내: "5명의 9월 생일 중 적어도 2명의 생일이 같을 확률이 얼마라고 생각하나요?"
- 슬라이더: 0% ~ 100% (1% 단위), 기본값 50% 위치
- 또는 직접 입력: 0~100 사이 정수
- "예측 완료" 버튼 클릭 시 Phase 2로 전환 (예측 후 수정 불가)

**Phase 2: 시뮬레이션**
- "시뮬레이션 실행" 버튼
- 1회 실행: 9월 1~30일 중 5개 날짜를 무작위 생성 → 겹치는 날짜가 있는지 표시
- 실행 단위 선택: [1회] [10회] [100회] 버튼
- **시각화**:
  - 상단: 현재까지의 시행 횟수 / "적어도 2명 같음" 발생 횟수 / 상대도수 (소수점 3자리)
  - 하단: 꺾은선 그래프 (x축: 누적 시행 횟수, y축: 상대도수 0~1)
    - **수학적 확률 기준선**: $\frac{1111}{3750} \approx 0.296$ 을 빨간 점선으로 항상 표시
    - **학생 예측값**: 파란 점선으로 표시
    - **시뮬레이션 결과**: 실선으로 실시간 갱신
- 시행 횟수가 늘어날수록 실선이 빨간 점선에 수렴하는 과정을 관찰
- 최대 시행: 1000회

**Phase 3: 계산 확인**
- 시뮬레이션 아래에 교과서 2단계 풀이를 STEP으로 표시:
  - "1단계: 5명의 생일이 모두 다를 확률"
  - $P(A^c) = \frac{{}_{30}P_5}{30^5} = \frac{30 \times 29 \times 28 \times 27 \times 26}{30^5}$
  - "= $\frac{17100720}{24300000} = \frac{2639}{3750}$"
  - "2단계: 여사건 → $P(A) = 1 - \frac{2639}{3750} = \frac{1111}{3750} \approx 0.296$"
- 이 풀이는 정답 확인 토글(`answer-reveal`)로 감싸서, 시뮬레이션을 충분히 관찰한 뒤 열도록 함

**Phase 간 전환 규칙**:
- Phase 1 완료 전 Phase 2 비활성
- Phase 2는 자유롭게 반복 가능 (리셋 버튼 제공)
- Phase 3(계산 확인)은 시뮬레이션과 독립적으로 언제든 열 수 있음

#### 시뮬레이션 엔진 규격
- `Math.random() * 30 + 1` → `Math.floor()` → 1~30 정수
- 5개 날짜 생성 → Set 크기가 5 미만이면 "겹침 발생"
- 시드 고정 없음 (매 실행 다른 결과)
- 성능: 100회 배치 실행은 즉시 완료되어야 함 (requestAnimationFrame 사용)

#### Chart.js 설정
- 타입: `line`
- x축: 누적 시행 횟수 (0~1000)
- y축: 상대도수 (0~1, 스텝 0.1)
- 기준선: `annotation` 플러그인 또는 수동 `beforeDraw`로 y=0.296 빨간 점선
- 예측값: y=(학생입력/100) 파란 점선
- 애니메이션: 꺼짐 (실시간 갱신이므로)

#### 예외 처리
- Phase 1에서 예측 없이 "시뮬레이션 실행" 시도 → "먼저 예측값을 입력해 주세요" 안내
- 1000회 초과 시도 → "최대 1000회까지 가능합니다" + 리셋 안내

## 8. 정답표 (검증용)

| 문제 | 정답 | 검증 |
|------|------|------|
| 기초UP 1 | $\frac{5}{12}$ | $\frac{1}{3} + \frac{1}{4} - \frac{1}{6} = \frac{4+3-2}{12}$ ✓ |
| 기초UP 2 | $\frac{1}{3}$ | $\frac{2}{3} - \frac{1}{2} + \frac{1}{6} = \frac{4-3+1}{6}$ ✓ |
| 기초UP 3 | $\frac{2}{5}$ | $3k = \frac{3}{5}$, $k = \frac{1}{5}$, $2k = \frac{2}{5}$ ✓ |
| 기초UP 4⑴ | $\frac{1}{4}$ | $\frac{5}{20}$ ✓ |
| 기초UP 4⑵ | $\frac{3}{20}$ | $\{6,12,18\}$ → $\frac{3}{20}$ ✓ |
| 기초UP 4⑶ | $\frac{1}{20}$ | $\{12\}$ → $\frac{1}{20}$ ✓ |
| 기초UP 4⑷ | $\frac{7}{20}$ | $\frac{5+3-1}{20}$ ✓ |
| 기초UP 5⑴ | $\frac{1}{3}$ | $\frac{{}_{6}C_2}{{}_{10}C_2} = \frac{15}{45}$ ✓ |
| 기초UP 5⑵ | $\frac{2}{3}$ | $1 - \frac{1}{3}$ ✓ |
| 스스로 1 | $\frac{5}{12}$ | $\frac{1}{2} + \frac{1}{4} - \frac{1}{3} = \frac{6+3-4}{12}$ ✓ |
| 스스로 2 | $\frac{2}{5}$ | $\frac{24}{60}$ ✓ |
| 스스로 3 | $0.65$ | $0.30 + 0.45 - 0.10$ ✓ |
| 스스로 4 | $\frac{11}{13}$ | $1 - \frac{56}{364} = 1 - \frac{2}{13}$ ✓ |
| 스스로 5 | $\frac{21}{22}$ | $1 - \frac{36}{792} = 1 - \frac{1}{22}$ ✓ |
| 생일 문제 | $\frac{1111}{3750}$ | $1 - \frac{17100720}{24300000}$, $30^5 = 24300000$, ${}_{30}P_5 = 17100720$ ✓ |
