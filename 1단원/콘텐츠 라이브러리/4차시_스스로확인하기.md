## 스스로 확인하기

### 문제 1
**지문**: 3개의 숫자 0, 1, 2에서 중복을 허용하여 만들 수 있는 네 자리 자연수에 대하여 다음을 구하시오.

⑴ 네 자리 자연수의 개수
⑵ 짝수인 네 자리 자연수의 개수

**풀이**:
- ⑴ 천의 자리: 1, 2 → 2가지, 나머지 각 자리: 3가지 → \(2 \times 3^{3} = 54\)
- ⑵ 짝수 → 일의 자리: 0 또는 2
  - 일의 자리 0: 천의 자리 2가지 × 중간 \(3^{2}\) = 18
  - 일의 자리 2: 천의 자리 2가지 × 중간 \(3^{2}\) = 18
  - 합: \(18 + 18 = 36\)

**정답**:
- ⑴ \(54\)
- ⑵ \(36\)

### 문제 2
**지문**: 6명의 학생이 코미디, 액션, 스포츠 영화 중 하나를 선택하여 관람할 때, 영화를 선택하는 경우의 수를 구하시오.

**풀이**: \({}_{3}\Pi_{6} = 3^{6} = 729\)

**정답**: \(729\)

### 문제 3
**지문**: success의 모든 철자를 일렬로 나열할 때, 양 끝에 모음 u와 e가 오는 경우의 수를 구하시오.

**풀이**:
- success: s(3), u(1), c(2), e(1) → 총 7자
- 양 끝에 u, e 배치: 2가지 (u__...e 또는 e__...u)
- 가운데 5자리에 s, s, s, c, c 나열: \(\dfrac{5!}{3!\,2!} = 10\)
- \(2 \times 10 = 20\)

**정답**: \(20\)

### 문제 4
**지문**: 오른쪽 그림과 같은 도로망에서 A 지점에서 B 지점까지 최단 거리로 갈 때, 두 지점 P, Q를 모두 지나는 경우의 수를 구하시오.

**풀이**:
- A→P 경로 × P→Q 경로 × Q→B 경로
- 구간별 계산 후 곱의 법칙 적용: \(3 \times 2 \times 2 = 12\)

**정답**: \(12\)

### 문제 5
**지문**: 집합 \(X = \{1, 2, 3\}\)에서 집합 \(Y = \{a, b, c, d\}\)로의 다음을 구하시오.

⑴ 함수의 개수
⑵ 일대일함수의 개수

**풀이**:
- ⑴ 각 원소에 Y의 4개 중 하나 대응: \({}_{4}\Pi_{3} = 4^{3} = 64\)
- ⑵ 중복 없이 대응: \({}_{4}\mathrm{P}_{3} = 4 \times 3 \times 2 = 24\)

**정답**:
- ⑴ \(64\)
- ⑵ \(24\)

## 수학으로 생각 넓히기

**지문**: 서로 다른 공 4개를 상자 A, B, C에 넣는 경우의 수를 구하려고 한다. 예나와 수호의 풀이를 살펴보고, 누가 바르게 풀었는지 판단하고 그 이유를 설명하시오.

**예나의 풀이**: 각 공이 3개의 상자 중 하나를 선택 → \({}_{3}\Pi_{4} = 3^{4} = 81\)

**수호의 풀이**: 각 상자가 4개의 공 중 하나를 선택 → \({}_{4}\Pi_{3} = 4^{3} = 64\)

**정답**: **예나**가 바르게 풀었다.
- 공의 관점: 각 공(4개)이 상자(3개) 중 하나를 선택 → \(3^{4} = 81\)
- 수호의 오류: 상자의 관점에서는 여러 공이 같은 상자에 들어갈 수 있으므로 "상자가 공을 하나 선택"하는 모델이 부적절하다.
