- **핵심 아이디어**: 사건이 일어날 수 있는 모든 경우를 분류하고 경우의 수를 체계적으로 세는 여러 가지 방법은 다양한 문제 상황에 활용된다.
- **소단원 구성**: 1. 여러 가지 순열 | 2. 중복조합 | 3. 이항정리
- **성취기준**:
  - [12확통01-01] 중복순열, 같은 것이 있는 순열을 이해하고, 그 순열의 수를 구하는 방법을 설명할 수 있다.

  - [12확통01-03] 이항정리를 이해하고, 이를 활용하여 문제를 해결할 수 있다.

### 차시 ② | 1. 여러 가지 순열 (1) - 중복순열
- **교과서**: 11~13쪽
- **학습 요소**: 중복순열, ₙΠᵣ
- 중복순열이란 무엇일까? (탐구)
  - 열기: 자전거 잠금장치의 비밀번호 (0~9 숫자, 세 자리) → 경우의 수 1000
  - 키우기: 서로 다른 n개에서 중복을 허용하여 r개를 택하여 나열하는 경우의 수
- 중복순열: n개에서 r개를 택하는 중복순열, 기호 ₙΠᵣ
- 중복순열의 수: ₙΠᵣ = nʳ
- 비주얼 개념: 2개의 문자 a, b에서 중복을 허용하여 3개를 택하는 중복순열 → ₂Π₃ = 2³ = 8
- 문제 1: ₆Π₃ = 216, ₄Π₄ = 256, ₇Π₂ = 49
- 문제 2: 바이올린과 피아노 중 하나를 택하여 7일 연습 일정표 → 128
- 예제 1: 4개의 숫자 0, 1, 2, 3으로 만들 수 있는 세 자리 자연수의 개수 → 48
  - 스스로 하기: 5개의 숫자 0, 1, 2, 3, 4로 만들 수 있는 네 자리 자연수의 개수 → 500
- 문제 3: 0~9 숫자로 네 자리 비밀번호 → (1) 10000, (2) 첫 자리 홀수 5000
- 문제 4: 5명이 방송반/댄스반 중 하나 선택 → (1) 32, (2) 각 동아리 적어도 1명 30

### 차시 ③ | 1. 여러 가지 순열 (2) - 같은 것이 있는 순열
- **교과서**: 14~15쪽
- 같은 것이 있는 순열의 수는 어떻게 구할까?
  - 1단계: a와 b를 각각 구별하여 나열하기
    - 2개의 a를 a₁, a₂로, 3개의 b를 b₁, b₂, b₃으로 구별하면 2!×3!개의 중복
  - 2단계: 같은 것이 있는 순열의 수 구하기
    - k × (2! × 3!) = 5! → k = 5!/(2!×3!) = 10
- 같은 것이 있는 순열의 수
  - n개 중에서 같은 것이 각각 p개, q개, ..., r개씩 있을 때: n!/(p!q!...r!) (단, p+q+...+r=n)
- 비주얼 개념: 4개의 숫자 1, 1, 2, 2를 사용한 네 자리 자연수의 개수 → 4!/(2!×2!) = 6
- 문제 5: banana의 모든 철자 일렬로 나열 → 60, mississippi → 34650
- 예제 2: A에서 B까지 최단 거리로 가는 경우의 수 → 6!/(4!×2!) = 15
- 문제 6: 집에서 도서관까지 최단 거리 → (1) 126, (2) 편의점 거쳐서 60

### 차시 ⑥ | 2. 중복조합 (1)
- **교과서**: 18~21쪽
- **학습 요소**: 중복조합, ₙHᵣ
- 중복조합이란 무엇일까? (탐구)
  - 열기: 초코컵케이크와 딸기컵케이크 중에서 3개를 사는 경우
  - 키우기: 서로 다른 n개에서 중복을 허용하여 r개를 택하는 경우의 수
- 중복조합: n개에서 중복을 허용하여 r개를 택하는 조합, 기호 ₙHᵣ
- 중복조합의 수 구하기: ●와 | 를 이용한 방법
  - ₃H₃는 3개의 ●와 2개의 | 를 모두 일렬로 나열하는 같은 것이 있는 순열의 수 → 5!/(3!×2!) = 10
  - 일반적으로 ₙHᵣ = {r+(n-1)}! / {r!(n-1)!} = ᵣ₊₍ₙ₋₁₎Cᵣ = ₙ₊ᵣ₋₁Cᵣ
- 중복조합의 수: ₙHᵣ = ₙ₊ᵣ₋₁Cᵣ
- 비주얼 개념: ₂H₆ = ₂₊₆₋₁C₆ = ₇C₆ = ₇C₁ = 7
- 문제 1: ₂H₄ = 5, ₆H₁ = 6, ₈H₆ = 462
- 문제 2: 토마토주스, 오렌지주스, 사과주스 중 중복 허용하여 8병 → 45
- 예제 1: (a+b+c)⁴의 전개식에서 서로 다른 항의 개수 → ₃H₄ = ₆C₄ = ₆C₂ = 15
  - 스스로 하기: (a+b+c)⁶의 전개식에서 서로 다른 항의 개수 → ₃H₆ = ₈C₆ = ₈C₂ = 28
- 문제 3: 서로 다른 항의 개수 → (1) (a+b+c+d)³ = 20, (2) (a+b)²(x+y+z)⁵ = 63
- 예제 2: 방정식 x+y+z=7에 대하여
  - (1) 음이 아닌 정수 x, y, z의 모든 순서쌍 → ₃H₇ = ₉C₇ = ₉C₂ = 36
  - (2) 자연수 x, y, z의 모든 순서쌍 → ₃H₄ = ₆C₄ = ₆C₂ = 15
- 문제 4: 방정식 x+y+z+w=9 → (1) 음이 아닌 정수 220, (2) 자연수 56
- 설명하기: 경우의 수를 각각 구하고 차이를 설명
  - (1) 서로 다른 색의 공 6개를 서로 다른 상자 3개에 → ₃Π₆ = 729
  - (2) 같은 색의 공 6개를 서로 다른 상자 3개에 → ₃H₆ = 28

### 차시 ⑧ | 수학+생활 채널 - 기명 투표와 무기명 투표

### 차시 ⑨ | 3. 이항정리 (1) - 이항정리와 이항계수, 파스칼의 삼각형
- **교과서**: 24~27쪽
- **학습 요소**: 이항정리, 이항계수, 파스칼의 삼각형
- 이항정리란 무엇일까? (탐구)
  - 열기: a, b가 적힌 공이 한 개씩 들어 있는 서로 다른 3개의 바구니에서 공 꺼내기
    - (1) a가 적힌 공 2개, b가 적힌 공 1개를 꺼내는 경우의 수 → ₃C₁ = 3
    - (2) (a+b)³의 전개식에서 a²b의 계수 → 3
  - 키우기: (a+b)ⁿ의 전개식에서 각 항의 계수
- 이항정리와 이항계수
  - (a+b)³ = a³ + 3a²b + 3ab² + b³
  - a²b의 계수는 세 개의 인수 (a+b) 중에서 b를 택할 인수 한 개를 뽑는 조합의 수 ₃C₁
- 이항정리: n이 자연수일 때
  - (a+b)ⁿ = ₙC₀aⁿ + ₙC₁aⁿ⁻¹b + ... + ₙCᵣaⁿ⁻ʳbʳ + ... + ₙCₙbⁿ
- 이항계수: ₙC₀, ₙC₁, ..., ₙCᵣ, ..., ₙCₙ
- 일반항: ₙCᵣaⁿ⁻ʳbʳ
- 비주얼 개념: (x-2)⁴를 이항정리로 전개 → x⁴ - 8x³ + 24x² - 32x + 16
- 문제 1: 이항정리를 이용하여 전개
  - (1) (a+3)⁴ = a⁴+12a³+54a²+108a+81
  - (2) (x-y)⁵ = x⁵-5x⁴y+10x³y²-10x²y³+5xy⁴-y⁵
  - (3) (2a+b)⁵ = 32a⁵+80a⁴b+80a³b²+40a²b³+10ab⁴+b⁵
- 예제 1: (x+3/x)⁶의 전개식에서 x²의 계수 → 135
  - 스스로 하기: (2x-1/x)⁵의 전개식에서 x의 계수 → 80
- 문제 2: (1) (x²-2/x)⁷에서 x⁵의 계수 → -280, (2) (4x-1/2x)⁸에서 상수항 → 1120
- 예제 2: 등식 ₙC₀+ₙC₁+ₙC₂+...+ₙCₙ = 2ⁿ 증명
  - (1+x)ⁿ에 x=1을 대입
- 문제 3: (1) ₙC₀-ₙC₁+ₙC₂-...+(-1)ⁿₙCₙ = 0 증명, (2) 짝수 번째 이항계수의 합 = 홀수 번째 이항계수의 합
- 파스칼의 삼각형
  - ₙCᵣ = ₙ₋₁Cᵣ₋₁ + ₙ₋₁Cᵣ (단, 1≤r<n)
  - 각 단계의 수는 그 위 단계의 이웃하는 두 수의 합
  - ₙCᵣ = ₙCₙ₋ᵣ이므로 좌우 대칭
- 비주얼 개념: 파스칼의 삼각형을 이용하여 다항식 전개
  - (a+b)³ = a³+3a²b+3ab²+b³
  - (x+1)⁴ = x⁴+4x³+6x²+4x+1
- 문제 4: 파스칼의 삼각형을 이용하여 전개
  - (1) (a+b)⁵ = a⁵+5a⁴b+10a³b²+10a²b³+5ab⁴+b⁵
  - (2) (x-y)⁶ = x⁶-6x⁵y+15x⁴y²-20x³y³+15x²y⁴-6xy⁵+y⁶

  5. ₂ₙC₀+₂ₙC₂+₂ₙC₄+...+₂ₙC₂ₙ = 2¹¹을 만족시키는 자연수 n → 6
  6. (x+2)¹⁵에서 xᵏ의 계수가 xᵏ⁺¹의 계수보다 크게 되는 자연수 k의 최솟값 → 5
- 수학으로 생각 넓히기: 파스칼의 삼각형에서 색칠한 부분의 합
  - ₄C₀+₄C₁+...+₈C₆의 합 → 119

### 차시 ⑪ | 수학+역사 채널 - 파스칼의 삼각형의 발견
- **교과서**: 29쪽
- 파스칼의 삼각형의 역사
  - 파스칼(Pascal, B., 1623~1662)이 쓴 『수삼각형론(Traite du Triangle Arithmetique)』에서 처음 소개
  - 중국의 주세걸(朱世傑, 1270?~1330?)은 1303년 출판된 『사원옥감』에서 파스칼의 삼각형을 소개
  - 파스칼의 삼각형을 '산술삼각형(arithmetic triangle)'이라 부르기도 함
  - 우리나라 조선시대 산학자 홍정하(洪正夏, 1684~?)가 쓴 『구일집(九一集)』에도 이항계수의 정리에 대한 내용이 담겨 있음
- 적용하기: 조선시대의 산학자 홍정하에 대하여 조사

### 차시 ⑫ | 스스로 정리하기 / 스스로 마무리하기 (1)
- **교과서**: 30~31쪽
- 스스로 정리하기 (p.30)
  - 여러 가지 순열
    - 중복순열: n개에서 중복을 허용하여 r개를 택하여 일렬로 나열하는 순열, 기호 ₙΠᵣ
    - 중복순열의 수: ₙΠᵣ = nʳ
    - 같은 것이 있는 순열의 수: n!/(p!q!...r!) (단, p+q+...+r=n)
  - 중복조합
    - 중복조합: n개에서 중복을 허용하여 r개를 택하는 조합, 기호 ₙHᵣ
    - 중복조합의 수: ₙHᵣ = ₙ₊ᵣ₋₁Cᵣ
    - 중복조합을 이용한 다항식의 항의 개수: (a+b+c)ⁿ의 전개식에서 서로 다른 항의 개수 = ₃Hₙ
    - 중복조합을 이용한 방정식의 해의 개수: x+y+z=n (n은 자연수)를 만족시키는 음이 아닌 정수해의 개수 = ₃Hₙ
  - 이항정리
    - (a+b)ⁿ = ₙC₀aⁿ + ₙC₁aⁿ⁻¹b + ... + ₙCᵣaⁿ⁻ʳbʳ + ... + ₙCₙbⁿ
    - 이항계수: ₙC₀, ₙC₁, ..., ₙCᵣ, ..., ₙCₙ
    - (a+b)ⁿ의 전개식의 일반항: ₙCᵣaⁿ⁻ʳbʳ
    - (1+x)ⁿ = ₙC₀ + ₙC₁x + ₙC₂x² + ... + ₙCₙxⁿ
    - 파스칼의 삼각형
    - 이항계수의 성질: ₙC₀+ₙC₁+...+ₙCₙ = 2ⁿ, ₙC₀-ₙC₁+...+(-1)ⁿₙCₙ = 0
- 스스로 마무리하기 (p.31): 문제 01~08 (객관식)
  - 01: ₄C₃+₂Π₂의 값 → ② 26
  - 02: 서로 다른 6개의 공을 3개의 주머니에 A에 2개만 넣는 수 → ④ 240
  - 03: 문자 a, b, c 중복 허용 5개 나열, 양 끝 서로 다른 문자 → ① 162
  - 04: 치역과 공역이 서로 같은 함수의 개수 → ④ 14
  - 05: 6개의 문자 a, b, c, d, e, f 일렬로 나열, a가 b보다 앞에 오고 f가 c보다 뒤에 오도록 → ④ 180
  - 06: 도로망 최단 거리 경우의 수 → ② 27
  - 07: 주머니 A에서 B로 공 옮기기 → ③ 8
  - 08: 같은 종류의 공 15개를 4상자에 2개 이상 → ④ 120

### 차시 ⑬ | 스스로 마무리하기 (2)
- **교과서**: 32~33쪽
- 스스로 마무리하기: 문제 09~19
  - 09: 5개의 숫자에서 중복 허용 6개를 택할 때 숫자 3을 2개 이상 → ② 70
  - 10: 방정식 a+b+c+d=17, 자연수, a, b, c 홀수, d 짝수인 모든 순서쌍 → ④ 84

### 차시 ⑭ | 창의 수학 프로젝트 / 미래 수학 유니버스
- **교과서**: 34~37쪽
- 창의 수학 프로젝트: 경주 문화 탐방 스탬프 투어 (p.34~35)
  - 경주의 권역별 역사 문화 명소 소개 (북부권, 서부권, 경주 시내권, 보문관광단지권, 불국사권, 남산권, 동해권)
  - 일정 1: 경주 시내권, 보문관광단지권, 불국사권, 동해권을 한 번씩 방문, 경주 시내권이 동해권보다 먼저
    - 같은 것이 있는 순열: 4개의 문자 a, b, c, a → 4!/2! = 12
  - 일정 2: 4개 권역에서 관광지 방문 스탬프 총 8개 찍기 (각 권역 적어도 1개)
    - 중복조합: a+b+c+d=8 (자연수) → ₄H₄ = ₇C₄ = ₇C₃ = 35
  - 프로젝트 과제: 모둠별로 다른 문화 관광지를 선택해 여행하는 방법과 경우의 수 구하기
- 미래 수학 유니버스: 쉽게 뚫리지 않는 양자 내성 암호 (p.36~37)
  - RSA 암호: 리베스트(1947~), 샤미르(1952~), 애들먼(1945~)이 1978년 발명
  - RSA 암호를 풀려면 매우 큰 자연수를 두 소수의 곱으로 소인수분해 해야 함
  - 양자 컴퓨터: 기존 컴퓨터보다 암호 해독 시간을 획기적으로 단축
  - 쇼어(Shor, W. P., 1959~)가 1994년 양자 컴퓨터를 이용한 소인수분해 알고리즘 개발
  - 양자 내성 암호(PQC: Post-Quantum Cryptography): 양자 컴퓨터로도 쉽게 뚫리지 않는 새로운 암호
  - 적용하기: 생성형 인공지능 프로그램을 이용하여 양자 내성 암호의 종류 조사
