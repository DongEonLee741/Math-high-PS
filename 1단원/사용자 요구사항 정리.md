# 일자 별 생성 시 요구 사항

## 개요
- 사용자 요구 사항은 사용자가 교과서 내용 이외에 추가로 생성을 원하거나 내용의 재구성을 요청할 때 작성함.
- 학습지 및 슬라이드 생성 시, 단원 내부 차시별 페이지 정리.md 파일을 읽고 본 문서를 다시 한번 더 읽은 후 pdf 교과서를 다시 한번 확인하고 제작할 것 .
- 완료된 차시는 일자 옆에 "(완료)" 라고 표시할 것.

## 2026.3.10 생성 요구
### 10차시
#### 생성 페이지 
- 28쪽, 30쪽 총 2페이지를 합쳐서 생성.

### 11차시
#### 생성 페이지
- 31쪽~33쪽 생성
#### 추가 요구 사항(기능)
1. 단원을 정리하여 문제를 푸는 페이지임
2. 문제가 많이 나오므로 학교에서 문제를 다 풀어주기에는 한계가 있음
3. 따라서 '투표' 시스템 생성
4. 학생이 문제를 풀때 설명이 필요한 것 같은 문제가 있으면 작성란 상단의 '물어보기' 버튼을 클릭함
5. 슬라이드 상단의 '현황보기' 버튼을 클릭하면 밑에 투표 현황을 확인할 수 있음
6. 이때 문제 별로 투표 해야 하기 때문에 투표를 문제 별로 할 수 있도록 제작

### 기능 수정 요구 사항

- 보내기 버튼을 작게 만들어 작성란의 펜 크기 조절 하는 곳 옆에 두기 
- UI도 이에 맞게 변경하기
- 11차시만 특별히 물어보기 버튼도 보내기 옆에 붙이기
#### 보내기 기능 변경
- 보내기를 '작성란' 별로 보낼 수 있게 변경
- 슬라이드에 보일 때도 이에 맞춰 변경
